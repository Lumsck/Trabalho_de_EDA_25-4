Sistema de gerenciamento de dados em **C** utilizando **Listas Simplesmente Encadeadas**. O código implementa alocação dinâmica e cobre todo o ciclo de operações CRUD (Inserir, Consultar, Atualizar e Remover). Desenvolvido para a matéria de Estrutura de Dados 1 na Universidade de Brasília (UnB), conduzida pelo professor: Lucas Ursulino Boaventura.
